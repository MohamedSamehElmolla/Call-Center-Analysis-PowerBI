# 📞 Call Center Performance Analysis

## 📌 Overview
This project analyzes call center performance using interactive dashboards built with Microsoft Power BI. It focuses on evaluating operational efficiency and agent performance based on large-scale data.

## 📊 Key Features
- Analysis of 1.7M+ call records
- KPI Tracking:
  - Total Calls
  - Handling Rate
  - Average Speed of Answer (ASA)
  - Service Level
- Agent performance evaluation
- Monthly trends and comparisons

## 🛠️ Tools & Technologies
- Power BI
- DAX
- Power Query

## 📈 Insights
- Identified top-performing agents
- Analyzed call handling efficiency
- Highlighted trends affecting service level and response time

## 🚀 How to Use
- Open the `.pbix` file using Power BI Desktop
